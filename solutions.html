Solutions

Learning Tapestry builds and operates infrastructure components with new capabilities on top of existing education systems. These components help provide snap-in digital content search and analytics functions in learning environments. These resources are released as open source to provide low-cost and modifiable software to plug into todayâ€™s content and learning management systems.

Examples include analytic plugins to help track student activity and present it to teachers in a dashboard, LTI integrations to connect learning managements systems with custom grading solutions, and embedded search for learning content.

Content discovery engine

The Content Discovery Engine (CDE) provides applications the ability to search through a vast collections of educational web sites, videos, documents and primary sources. This engine allows for ingestion of learning content metadata in LRMI format and provides APIs as a searchable interface for content and learning management systems.

The Analytics Engine collects learning events such as page views, page clicks and time on videos from whitelisted sources. Once collected, the Analytics Engine provides API interfaces to retrieve statistics by resource type, URL and time period.
